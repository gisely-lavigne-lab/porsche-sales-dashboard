# 🏁 Porsche Sales Executive Performance Dashboard

Um dashboard executivo interativo de alta performance desenvolvido para análise granular de vendas de veículos Porsche. A interface foi projetada seguindo a identidade visual premium da marca (Clean Black, Guards Red e Racing Gold), utilizando dados sanitizados de vendas reais e projeções.

## 📊 Funcionalidades Clave

* **Filtros Avançados Dinâmicos:** Filtre instantaneamente todo o ecossistema do dashboard por Modelo de Porsche, Ano do Modelo, Cidade e Método de Pagamento.
* **Carrossel de Heróis Interativo:** Seleção rápida de modelos topo de linha (`911 Turbo S`, `Cayenne Coupé`, `Panamera Turbo S`) diretamente através de imagens integradas.
* **Métricas em Tempo Real (KPIs):**
    * Faturamento Total Consolidado.
    * Unidades Totais Vendidas.
    * Ticket Médio por Veículo.
    * Líder de Volume de Vendas.
* **Insights Estratégicos Automatizados:** Geração automática de inteligência de negócios baseada nos filtros aplicados (ex: performance regional e anos-modelo de maior sucesso).
* **Visualização Gráfica Avançada:** Gráficos interativos renderizados via Chart.js contendo volumes por ano, popularidade regional empilhada e distribuição por cidade.
* **Tabela Granular Sanitizada:** Listagem completa dos registros de venda com badges dinâmicos de status de entrega (`Delivered`, `In Transit`, `Pending`, `Cancelled`).

## 🛠️ Tecnologias Utilizadas

* **HTML5** & **CSS3** (Variáveis CSS, CSS Grid Layout, Flexbox e design responsivo).
* **JavaScript (ES6+)** para manipulação de dados em memória, cálculo de KPIs e reatividade dos filtros.
* **Chart.js** (via CDN) para a renderização de gráficos vetoriais de alta performance.
* **Inter Font Family** para tipografia limpa e moderna.
